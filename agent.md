# VS Code Agent план (Copilot/Agents)

## Цілі
- [ ] Додати повний CRUD для ContentBlock/Project/Meeting/Transaction
- [ ] Реалізувати сторінку «Фінанси» для всіх учасників: баланс = Σ(income) - Σ(expense)
- [ ] Додати розділ «Збори засновників»: створення/редагування, порядок денний, голосування, автопротокол (markdown/PDF)
- [ ] Додати форму анкетування (12 питань) збереження у БД
- [ ] Пагінація/пошук для списків
- [ ] Завантаження CSV по транзакціях (імпорт банківських виписок)
- [ ] Email-підтвердження реєстрації (пізніше)
- [ ] Захист адмінки CSRF/рольові middleware (вже частково)
- [ ] Тести (pytest)

## Контекст для Copilot
- Фреймворк: Flask 3, SQLAlchemy 2
- БД: PostgreSQL, схема з `.env` -> `DB_SCHEMA`
- Деплой: Render (render.yaml, Procfile)
- Ключі: .env (не комітити)
- I18N: простий перемикач у сесії, поля *_uk/_de

## Пріоритети
1. Прозора фінансова сторінка (read-only для всіх членів)
2. Голосування за проєкти (вже є базово) + підрахунок
3. Збори засновників (UI + протокол)
