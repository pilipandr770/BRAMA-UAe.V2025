# e.V. Platform (Flask + PostgreSQL + Render)

## Швидкий старт
1. Створи `.env` з `.env.example`.
2. `pip install -r requirements.txt`
3. `python -m app.app` (перший запуск — створить схему та таблиці, адміна)
4. `flask run`

## Ролі
- `member` — користувач за замовчуванням
- `admin` — адміністратор (створення/редагування контенту, фінанси, ролі)
- `founder` — учредитель (доступ до сторінок зборів, голосування по порядку денному)

## Мовність
- Перемикач `?lang=uk|de` зберігається в сесії.
- Усі поля контенту дубльовані: *_uk / *_de.

## Зображення
- Галерея зберігає фото в PostgreSQL як `bytea`.
